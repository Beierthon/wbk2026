# Vercel alias: `apps/baustellen-tool`

The Vercel project **baustellen-tool** uses Root Directory `apps/baustellen-tool`.
That path is a symlink to [`apps/web`](../web) so both project names deploy the same
Next.js app without duplicating source.

Do not add a real `apps/baustellen-tool/` directory — it would shadow the symlink.
